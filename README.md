# Student Manager

A beginner Python project created to practice Python programming concepts.

 ##Features

-Add a student
-View all students
-Calculate class average
-Find a student by name
-Select a random "Student of the Day"
-Save and load student data using CSV files

## Technologies Used

- Python 3
- CSV module
- Random module
- Pytest (for testing)

## Project Structure

```
student-manager/
│── main.py
│── grades.py
│── test_grades.py
│── students.csv
│── README.md
```

## How to Run

```bash
python main.py
```
